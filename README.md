# Snake Game

## Overview
This is a simple Snake Game built using HTML, CSS, and JavaScript. The game features a grey-themed design and is centered on the page. Players can control the snake to collect food, with a real-time score display and options to pause, resume, or quit the game.

## Project Structure
```
snake-game
├── index.html      # The main HTML file containing the game structure
├── style.css       # The CSS file for styling the game
├── script.js       # The JavaScript file containing the game logic
└── README.md       # This README file
```

## How to Play
1. Open the `index.html` file in a web browser.
2. Use the arrow keys to control the direction of the snake.
3. Collect the food to increase your score.
4. Click the pause button to pause the game.
5. Click the resume button to continue playing.
6. Click the quit button to end the game.

## Running the Game
To run the game, simply open the `index.html` file in any modern web browser. No additional setup is required.

## Design and Functionality
- The game area is set to 800px by 800px and is centered on the page.
- The grey theme provides a pleasing visual experience.
- The score is displayed in real-time, updating as the player collects food.
- The game includes intuitive icons for pause, resume, and quit functionalities.

## Additional Notes
Feel free to modify the styles in `style.css` or the game logic in `script.js` to enhance your gaming experience or to add new features. Enjoy playing!
